# Certificate System - Quick Usage Guide

## Quick Start

### 1. Register as a New Citizen
1. **Login Page** → Click "Register as Citizen" button
2. **Step 1 - Personal Info**
   - Name: Enter full name
   - Email: Valid email address
   - Phone: 10-digit mobile number
   - DOB: Must be 18+ years old
   - Gender: Select your gender
3. **Step 2 - Address & Security**
   - Aadhar: 12-digit Aadhar number
   - Address: Full address
   - Village: Village name
   - Password: Min 8 characters
4. **Click "Complete Registration"**
5. ✅ Automatic login - Ready to apply!

---

## 2. Apply for a Certificate (Citizen)

### Steps:
1. **Dashboard** → Click on "Certificates" in navigation
2. **Click "New Application"** button (top right)
3. **Fill the Form**:
   - **Personal Info** (Auto-filled, can edit):
     - Name, Email, Phone, Age
   - **Certificate Type**: Select from dropdown
     - Residence Certificate
     - Birth Certificate
     - Death Certificate
     - Marriage Certificate
   - **Document Upload**:
     - Upload at least one document:
       - Birth Certificate
       - Death Certificate
       - Residence Certificate
       - Marriage Certificate
       - (Optional) Photo
     - Click on document box → Select file (PDF/JPG/PNG)
     - ✅ Shows "Uploaded" when done
   - **Remarks** (Optional): Add any additional info

4. **Click "Submit Application"**
5. ✅ Application submitted with "Pending" status

---

## 3. Track Application Status (Citizen)

### View All Applications:
1. **Certificates** page shows all your applications
2. **Status Colors**:
   - 🟡 **PENDING** - Just submitted, awaiting staff review
   - 🔵 **UNDER REVIEW** - Staff is checking documents
   - 🟢 **APPROVED** - Approved! Can download
   - 🔴 **REJECTED** - Rejected, see reason to reapply

### View Details:
1. Click on any application row → Expands to show:
   - Full phone number
   - Your age
   - All uploaded documents
   - If rejected: Rejection reason
   - Remarks you left

### Download Certificate:
1. **Status must be "APPROVED"**
2. Click the **Download button** (⬇️ icon)
3. Certificate PDF downloads automatically

### Reapply if Rejected:
1. **Click "New Application"** again
2. Submit improved application
3. Staff reviews again

---

## 4. Review Applications (Staff/Admin)

### Access Review Mode:
1. **Staff/Admin Login** → Certificates
2. **Filter by "Under Review"** to see pending applications
3. Click on application to open **Review Modal**

### Review Process:
1. **Examine Details**:
   - Applicant info (Name, Email, Phone, Age)
   - Certificate type and dates
   - All uploaded documents

2. **Make Decision**:
   - **APPROVE**:
     - Click "Approve" button
     - Optionally add remarks
     - Click confirmation checkbox
     - Click "Confirm & Approve"
     - ✅ Certificate approved
   
   - **REJECT**:
     - Click "Reject" button
     - Enter detailed reason (mandatory)
     - Click confirmation checkbox
     - Click "Confirm & Reject"
     - ✅ Applicant sees reason, can reapply

## Status Explanations

### 🟡 PENDING
- Application just submitted
- Waiting for staff to review
- Typically 1-3 days

### 🔵 UNDER REVIEW
- Staff is currently reviewing
- Documents being verified
- Sending soon to approval

### 🟢 APPROVED
- ✅ Certificate approved!
- Download link available
- Valid for 4 years

### 🔴 REJECTED
- ❌ Application not approved
- Review the rejection reason
- Fix issues and reapply
- No additional fees for reapplication

## Troubleshooting

### Can't Upload Documents?
- Check file format (PDF, JPG, PNG only)
- File size under 5MB?
- Click on correct document type box
- Try different browser

### Status Not Updating?
- Refresh the page (F5)
- Clear browser cache
- Logout and login again

### Can't Download Certificate?
- Status must be "APPROVED"
- Check internet connection
- Try different browser
- Contact support if still fails

### Lost Password?
- (In production) Click "Forgot Password"
- For demo: Create new account

### Form Won't Submit?
- Fill all required fields (marked with *)
- At least 1 document must be uploaded
- Check for red error messages
- Try again

## FAQs

**Q: How long does approval take?**
A: Typically 1-3 working days depending on volume.

**Q: Can I upload multiple documents?**
A: Yes, upload all 4 types for complete verification.

**Q: What if I made a mistake?**
A: If not approved yet, you can delete and reapply. If rejected, reapply with corrections.

**Q: Is my data secure?**
A: Yes, encrypted storage and secure authentication.

**Q: Can I get a refund if rejected?**
A: No application fees in this demo. In production, refer to refund policy.

**Q: How many certificates can I apply for?**
A: Unlimited. Apply for as many as needed.

**Q: Is English the only language?**
A: Currently English. Multi-language coming soon.

**Q: Can I apply offline?**
A: No, internet connection required. Offline mode coming in future.
